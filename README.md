# Criando jogo simples snake com Python 

## Bibliotecas usadas:

•Pygame

Para instalar os requirements use:
````
pip install -r requirements.txt
````

Para baixar o repositório use:
````
git clone https://github.com/AlexandeOliveira12/Game-Snake.git
````
